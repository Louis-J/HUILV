有时当需要查询汇率时，常常难以快速找到合适的途径，本项目就是为了方便在偶然突发情况下，快速查询实时汇率和一段时间内汇率变化的小工具，打开微信搜索小程序即可使用。项目暂未开发完成上线，敬请期待。
